任务3：
    做一个商业活动文本二分类模型
任务背景：
    公司业务需要提取人物关系网络，设置了一些规则，比如，如果一些人在某一商业活动中同时出现，则认为这些人互相认识。在过滤垃圾文本之后，需要筛选出商业性质的文章，进而提取文章中的人物关系
解决方案：
    1.首先从es中按照一些带有商业性质的主题词，导出一些数据，在导数据时，就划分了正负样本，正样本为文本中出现至少三个不同人名的完整的关系，即（公司，职称，人名），这一划分规则已由小领导实现，准确率大概在70%+；负样本为文本中出现的关系少于两个，有可能人名重复，即这种情况，“xx公司联合创始人兼CTO张三”，就会被记录两次（xx公司，联合创始人，张三），（xx公司，CTO，张三）；正负样本比例约1：1，各大约4千条；
    2. 对正负样本也进行分词，去停用词，去特殊符号，但由于此次任务的关键特征是名词性的词（公司，职称，人名都是名词性质的），因此样本中只保留名词；
    3.接着利用tfidf提取特征，svm训练模型
测试结果：
    对es中大概30w+条文本做测试，预测正样本占约20%，负样本约为80%，比例符合预期。通过对结果分析，发现有一些文章人为判定时也是模棱两可，因为模型针对这些文本表现的不太好。
